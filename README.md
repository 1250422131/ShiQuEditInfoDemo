## 第三方库

- [KongzuDialogX](https://github.com/kongzue/DialogX)
    - 自定义对话框
    - 底部菜单对话框
    - 日期选择对话框
- [Lottie](https://lottiefiles.com/)
    - 加载动画
- Lifestyle-ViewModel-Ktx
    - 提供ViewModel中的生命周期安全的协程作用域
- Lifestyle-Fragment-Ktx
    - 提供ViewModel委托绑定能力
